# Responsive Landing Page

## 📌 Project Overview

This project is a fully responsive landing page developed using HTML, CSS, and JavaScript. The landing page features an interactive navigation menu that changes style when the user scrolls or hovers over menu items. The design adapts seamlessly across different screen sizes, providing an optimal user experience on desktops, tablets, and mobile devices.

## 🚀 Features

* Responsive design for all screen sizes
* Fixed navigation menu
* Navigation bar changes style on scroll
* Hover effects on menu items
* Modern and clean user interface
* Smooth scrolling experience
* Cross-browser compatibility

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
